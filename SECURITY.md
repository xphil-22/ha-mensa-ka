# Security Policy

If you discover a security issue in this repository, please do not open a public issue first.

Instead, contact the maintainer directly and include:

- a short description of the issue
- affected versions or commits
- reproduction steps
- expected impact

This project is a Home Assistant custom integration and depends on public upstream meal-plan data. At the moment, it does not store secrets for the upstream API because it uses read-only unauthenticated requests.
